## עדכון אחרון לפי ההערות
- היומנים סגורים כברירת מחדל. לחיצה על שם הקבלן פותחת/סוגרת את הימים.
- לכל יומן חודשי נוסף סיכום סופי:
  - סה״כ ימי עבודה
  - סה״כ הופעות פועלים
  - סיכום ימים לפי כל פועל
  - מקומות עבודה ללא כפילות
  - סכום משוער
- שם קבלן: בחירה מהירה "ויסאם" או "אחר / הוספה ידנית".
- אם בוחרים קבלן ידני ושומרים יום עבודה — נפתח לו יומן אוטומטי לפי קבלן+חודש+פרויקט.
- נוספה שמירה אוטומטית בזמן עריכה קיימת.
- ייצוא Excel:
  - לפי הסינון
  - יומן ספציפי של קבלן
  - קובץ כללי לכל הקבלנים לפי חודש/פרויקט


## עדכון קבלנים במקביל + חזור פעולה
- אפשר להוסיף כמה קבלנים במקביל באותו אתר ובאותו חודש.
- כל יום עבודה נשמר לפי קבלן + חודש + פרויקט, ונכנס אוטומטית ליומן השייך לו.
- נוספה רשימת קבלנים דינמית: ויסאם + כל קבלן שהוזן בעבר + אחר/הוספה ידנית.
- נוספו כפתורים:
  - שמור והוסף קבלן/יום נוסף
  - קבלן נוסף במקביל
- כדי למנוע החלפה בטעות, שמירה אוטומטית פועלת רק בעריכת יום קיים, לא ברשומה חדשה.
- אם משנים קבלן/חודש/פרויקט בזמן עריכת יום קיים, המערכת שואלת אם ליצור יום עבודה חדש במקום להחליף את הקיים.
- נוסף כפתור "חזור פעולה" לשחזור מחיקה/שמירה/אישור/שינוי סדר.
